# DivaZaap

## About

Simple multiplatform software to emulate the Dofus Unity <-> Ankama Launcher Apache Thrift protocol

Can be used to connect to any auth server with your own token.

Useful to make your own private server.

## Wails

This software is made with Wails. Run `make install-wails` to install Wails. You can then run with `make dev`.
